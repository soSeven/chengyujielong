# 成语接龙游戏

基于`Swift`语言，架构为`MVVM`，采用主要第三方库为 `RxSwift` + `Swinject` + `Moya`



<img src="demo.gif" height="400px" />



## 安装方法

```shell
git clone & pod install
```
